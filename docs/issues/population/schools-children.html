子どもが半減した — 学校再編と教育の質

#少子化 #学校統廃合 #教育

一言でいうと

横須賀市で生まれる子どもは、1995年の3,665人から2020年の2,025人へほぼ半減しました[1]。学校の統合が始まり、2025年に4校が2校になりました[3]。寂しさをこらえて統合するか、小さな学校を守るか。選択の時です。

💡 たとえると…生まれる赤ちゃんは、1日あたり10人から5人へ。まちの「新しい仲間」が毎日5人ずつ減った計算です。

いま何が起きている?

出生数は1995年の3,665人から2020年には2,025人へ、44.7%減りました[1]。2021年はさらに約1,809人です[1]。合計特殊出生率(1人の女性が産む子どもの数の目安)は1.18前後。全国(1.33)・神奈川県(1.26)を下回ります[1]。

15歳未満の人口は約3.7万人。10年前より約1.2万人、率にして約25%減りました[1]。同じ教室なら、机の4つに1つが空いた計算です。

学校も動き始めました。市は「教育環境整備計画」(2022〜2029年度)で再編を進めています。小学校の適正規模は12〜24学級とされています[2]。2025年4月には「田浦小+長浦小」「走水小+馬堀小」の2組4校が統合しました[3]。2026年3月には、富士見小・武山小・荻野小・大楠小・鴨居小・小原台小の6校が新たに再編対象に加わりました[4]。

なぜこうなった?

  • 子どもを産む年代の女性(15〜49歳)が、1990年の108,571人から2020年には68,135人へ減りました[1]。親になる世代そのものが減っています。
  • 出生率も1.18前後と、全国・県平均を下回ります[1]。
  • 子育て世代の市外流出が長く続きました。2013年には転出超過が全国の市町村でワースト1位でした[5]。
  • 子どもが減れば学級数が減り、小規模化した学校が適正規模[2]を保てなくなる、という連鎖です。

放っておくとどうなる?

5年後: 再編対象の6校[4]の行方が決まる時期です。話し合いを先送りすれば、小規模化だけが進みます。市の計画が示す適正規模[2]を下回る学校が増え、クラスの選択肢や部活動・行事の幅が狭くなる恐れがあります。

10年後: 出生数の減少[1]は、10年後の児童数をほぼ確実に決めます。準備なき統合の繰り返しは、子どもにも地域にも負担です。逆に、計画的に再編すれば、跡地を地域の財産として活かす時間がつくれます。

🛡 守りの一手 — 暮らしを支える・悪化を止める

  • 統合で通学が遠くなる子への通学支援。子どもの安全と保護者の負担軽減を統合の条件にします。
  • 学校跡地を地域に返す。避難所・集会・子育ての場として使い続けられる形を、閉校前に決めておきます。
  • 統合までの間も、小規模校の教育の質を落とさない。教員配置と学校間交流で支えます。

⚔ 攻めの一手 — 伸ばす・新しくする

  • 統合を「我慢」でなく「新しい学校づくり」の機会に。ICT(情報技術)活用や地域と連携した学びを、統合校から先行導入します。
  • 「子育てするなら横須賀」ブランドの発信。0〜18歳の医療費無料[6]、中学校完全給食(2021年開始)[7]、2026年度からの小学校給食費無償化[8]を束ねて伝えます。
  • 追い風を活かす。2025年は0〜14歳が245人の転入超過でした[9]。子育て世帯に選ばれ始めた今こそ、教育の魅力で定着させます。

どう進める?

  1. 短期(1〜2年)
    再編対象6校[4]の地域協議を丁寧に進める。通学支援と跡地の使い方を「統合とセット」で提示する。
  2. 中期(3〜5年)
    統合校で新しい学校モデル(ICT・地域連携)を形にする。跡地の地域利用を順次スタートする。
  3. 長期(10年)
    出生数の動向[1]に合わせて、計画(〜2029年度)[2]の次を描く。学校配置と地域の拠点づくりを一体で設計する。

⚖ トレードオフ — 全部はできない

地域から学校が消える寂しさ。学校は避難所であり、お祭りや見守りの拠点でもある
vs
子どもにとって適正規模の学びの環境。友だち・先生・行事の選択肢を確保する

「学校がなくなれば地域が衰える」という心配は、もっともです。学校は教育の場であると同時に、地域の集まりや防災の拠点だからです。一方で「子どものための学校なのだから、子どもの環境を最優先に」という声も、もっともです。このページは、どちらかを切り捨てるのではなく「学校機能は統合し、地域機能は残す」という分け方で両立を目指します。

★ オススメの構成

このサイトの提案は「再編は進める。ただし跡地を地域に返すことをセットにする」です。子どもには適正規模[2]の学び場を用意する(持続可能性)。地域には避難所・集会・子育て拠点としての跡地利用を約束する(QOL)。守り(通学支援・跡地の地域利用)×攻め(統合を機にした新しい学校づくり・「子育てするなら横須賀」発信[6][8])の組み合わせです。統合の痛みを地域への投資で返すことが、合意への近道だと考えます。

どう実現する?

お金(財源)

学校の統合で浮く維持費を、通学支援と跡地活用に優先的に回します。給食費無償化は月額5,700円相当を国の補助500円+市の負担5,200円で支える構造です[8]。継続には財源の確保が前提になります。

人(体制)

教育委員会だけで抱えない体制が必要です。跡地利用は防災・福祉・市民協働の部署と、地域住民が同じテーブルで決めます。

合意づくり

「統合ありき」で進めれば反発を招きます。児童数のデータ[1]と選択肢を先に示し、通学支援と跡地の使い方を地域と一緒に決める手順を守ります。
