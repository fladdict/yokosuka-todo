基地とともにある経済 — 安定雇用と依存リスクのあいだ

#米軍基地 #自衛隊 #雇用 #基地経済

一言でいうと

米軍と自衛隊の施設は、合わせて市の面積の約6.1%を占めます[1]。基地で働く日本人は5,470人で、市内最大級の安定した職場です[2]。ただし基地のあり方は国が決めるもので、市は直接コントロールできません。

たとえるなら、家計の大きな柱が「転勤のありうる勤め先」一本に寄っている家のようなものです。いまの給料は安定していても、人事を決めるのは本人ではなく会社。だから収入の柱は複数あったほうが安心です。

いま何が起きている?

横須賀市内の米軍施設は約336万平方メートルで、市域の約3.3%。自衛隊は37施設・約279万平方メートルあります。合計すると約615万平方メートル、市の土地のおよそ16分の1(約6.1%)が基地です[1]。

雇用の面では、市内の米軍施設で働く日本人従業員は5,470人(2025年3月31日現在)。うち5,307人が横須賀海軍施設で働いています。この数はゆるやかに増える傾向にあります[2]。なかでも米海軍の艦船修理廠(かんせんしゅうりしょう。SRF-JRMC、艦船を直す工場)は約2,500人の日本人を雇用しており[3]、市内最大級の「ものづくりの職場」でもあります。

一方、お金の面の思い込みには注意が必要です。基地があることで国から市に入る基地交付金と調整交付金は、2025年度で合計23.61億円。市の当初予算に対する比率はわずか1.3%です[4]。100円の予算のうち1円ちょっと。「基地のおかげで市の財政が潤っている」とは言えない規模です。参考として、米海軍横須賀地区の発注は2017年度見込みで工事契約86億円・サービス契約25億円という資料があります[5](数字はやや古い点に注意)。

そして大前提として、基地の規模や役割は国の安全保障政策で決まります。市にとっては、自分では動かせない「外部要因」です。

なぜこうなった?

  • 横須賀は港に適した地形をもち、1897年創業の浦賀船渠(うらがせんきょ)など、造船・海軍とともに産業が育ったまちだった[6]。戦後はその土地と機能が米軍・自衛隊に引き継がれた
  • 製造業の撤退が続いた結果(浦賀工場の2003年閉鎖[6]、日産追浜工場の2027年度末生産終了予定[7]など)、市の雇用に占める基地関連の存在感が相対的に増している
  • 基地交付金は税の代わりという性格の限られたお金で、市の財政を支える規模にはなっていない[4]

放っておくとどうなる?

5年後: 艦船修理を支える技能職[3]の採用・育成を市内でつなぐ仕組みがないと、せっかくの安定雇用に市民が就きにくくなります。基地の求人と地元の若者・訓練機関が結びつかないまま、雇用の機会を市外の人に譲ることになりかねません。

10年後: 国際情勢や国の方針が変われば、基地関連の雇用や発注は市の意思と無関係に増減します。増えた場合でも、受け皿となる地元人材と企業が育っていなければ恩恵を取りこぼします。減った場合、ほかの経済の柱が育っていなければ、まちは打つ手のないまま影響を受けます。どちらに転んでも「柱が一本だけ」の状態が一番もろいのです。

🛡 守りの一手 — 暮らしを支える・悪化を止める

  • 技能人材のパイプラインづくり。工業高校や職業訓練と、艦船修理廠(約2,500人[3])など基地内の技能職の採用を接続する。安定雇用[2]への入口を市民に開く
  • 造船・艦船修理の技能継承。市内の造船業が縮小するなかで(住友重機械の新造船撤退[8]など)、修理の現場に残る技能を次世代へつなぐ仕組みを支援する

⚔ 攻めの一手 — 伸ばす・新しくする

  • YRP(横須賀リサーチパーク)の再定義。1997年開設の情報通信研究拠点で、68社が入居(2026年6月現在)[9]。通信に加え、防衛テック・サイバーセキュリティ人材の育成拠点へ役割を広げる。近年は水素や半導体など通信以外の進出も続いており[10]、土壌はある
  • 軍港というまちの個性を観光に活かす。2024年の入込観光客は前年比141万人増(+15.8%)で過去最高[11]。ほかのまちにない「軍港の風景」を、観光・にぎわいの資源として磨く

どう進める?

  1. 短期(1〜2年)
    基地雇用の実態(職種・採用ルート)を市民に見える形で整理。工業高校・職業訓練と基地側の採用をつなぐ協議を始める。
  2. 中期(3〜5年)
    技能人材パイプラインを定着させる。YRPで防衛テック・サイバー分野の企業・講座を誘致し、「基地以外の柱」を太くする。
  3. 長期(10年)
    基地・研究開発・観光・スタートアップの複数の柱で経済を支える形を完成させ、国の方針変更に揺らがない構造をつくる。

⚖ トレードオフ — 全部はできない

基地関連をさらに深める(防衛需要を取り込み、安定雇用を太くする)
vs
依存リスクを分散する(基地以外の経済の柱に資源を振り向ける)

このテーマは、市民の間でも考え方が分かれます。深化を支持する立場には理があります。5,470人の雇用[2]は長年の実績であり、防衛省による追浜工場跡地の取得案(未決定・2026年6月時点)[12]のように、国の需要を取り込む機会も現にあります。確実な雇用を遠ざける理由はない、という考え方です。一方、分散を重視する立場にも理があります。基地は市が決められない外部要因であり、国際情勢ひとつで前提が変わりえます。また、基地の存在そのものへの賛否も市民の中にあり、基地一辺倒のまちづくりには同意できないという声も尊重されるべきです。両者が共有できる事実は二つあります。基地交付金は予算の1.3%[4]にすぎず「基地=財政の柱」ではないこと。そして、どんな立場でも「経済の柱が一本しかない状態はもろい」ことです。

★ オススメの構成

このサイトの推奨は「ポートフォリオ型」です。ポートフォリオとは、資産を一つに集中させず複数に分けて持つ考え方のこと。基地はすでにあるまちの前提として、技能人材パイプラインで安定雇用[2][3]への入口を市民に開きます(守り=QOL)。同時に、YRPの再定義[9][10]や観光[11]、スタートアップ支援など、基地以外の柱を複数育てます(攻め=持続可能性)。基地を「活かす」ことと「依存しない」ことは、両立できます。むしろ両立させることが、賛否の分かれるこのテーマで市民の合意をつくる現実的な道だと考えます。

どう実現する?

お金(財源)

基地交付金は予算の1.3%[4]と限られるため、これを当てにした計画は立てません。人材育成は既存の教育・訓練機関との連携を軸にし、新規の大型支出を抑えます。

人(体制)

市・教育機関(工業高校・職業訓練)・基地側の三者で、採用と育成をつなぐ常設の連絡体制をつくります。YRP関連は産業部門が一体で担当します。

合意づくり

基地への賛否が分かれることを前提に、まず「雇用は5,470人[2]、交付金は予算の1.3%[4]」という事実を共有するところから始めます。立場が違っても、データの土台がそろえば議論は前に進みます。
