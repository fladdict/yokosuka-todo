# 文章ルール(STYLE GUIDE)

このサイトの読者は**ふつうの市民**。行政や政策の専門家ではない。
中学3年生が読んで理解できる文章を基準にする。

## 必ず守ること

1. **難しい言葉を使わない**
   - ✗ 経常収支比率が高止まりし財政の硬直化が進行
   - ○ 市の家計にたとえると、毎月の給料がほぼ固定費で消えて、自由に使えるお金がほとんど残らない状態
   - 専門用語をどうしても使うときは、直後にカッコで一言説明を入れる

2. **メタファ(例え話)と具体例を入れる**
   - 各課題の「一言でいうと」には必ず例え話を1つ
   - 抽象的な話には「たとえば○○さんの場合…」のような具体例を添える

3. **長い文は分解する**
   - 1文は60字以内を目安。「〜であり、〜だが、〜なので」と続けない
   - 1段落は3〜4文まで

4. **専門性は外部リンクへ逃がす**
   - 本文では結論と意味だけ書く。詳細な制度・統計手法・計画書は「もっと知りたい人へ」セクションの外部リンクに誘導
   - リンクには「何がわかるか」を一言添える(例:「市の人口推計の元データ(横須賀市・PDF)」)

5. **数字は実感に変換する**
   - ✗ 人口が年間約3,000人減少
   - ○ 1年で小学校5〜6校分の人がまちから減っている計算

6. **おどさない、でもごまかさない**
   - 不安をあおる表現は使わない。一方で、悪いデータを隠したり薄めたりしない
   - 「困りごと」と「打ち手」を必ずセットで示す

7. **両論併記**
   - トレードオフのセクションでは、賛成側・反対側のどちらの立場の人が読んでも「自分の意見も書かれている」と感じる書き方をする

8. **出典のない数値は書かない**
   - すべての数値は出典(できれば一次資料)つき。確認できなければ載せない

9. **文体はです・ます調に統一する**
   - ✗ 〜が進行している。〜が必要だ。
   - ○ 〜が進んでいます。〜が必要です。
   - 見出し・表・箇条書きの体言止めは可。本文の地の文はです・ます調で書く

10. **空疎動詞の禁止**
   - 「推進する」「強化する」「開拓する」「ブランド化する」等で施策を終わらせない
   - **誰が・何を・どの制度や手段で**を最低1つ添える
   - ✗ 観光をブランド化し、にぎわいを強化する
   - ○ 市の観光課が、宿泊税(検討中)を財源に、ナイトタイムのガイドツアーを年10本企画する
