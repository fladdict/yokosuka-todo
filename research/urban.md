## 1. 谷戸(やと)地域

- 谷戸地域は丘陵の谷あいに階段・狭隘路で形成された住宅地。**市内の空き家の約8割が「階段地(車が入れない区域)」に存在。車を横付けできない物件の空き家率は45.7%**(市調査の引用) / https://hamarepo.com/story.php?story_id=4455 / △(市資料の引用。一次確認推奨)
- 市は2012年度から汐入町5丁目2区をモデル地区に「谷戸地域住環境対策」を開始。2015年に汐入駅・逸見駅から約500m圏で「車道から階段約40段以上」の区域へ拡大 / https://www.city.yokosuka.kanagawa.jp/4821/yatobank/test-top.html / ◎
- 対策メニュー: **谷戸空き家バンク、老朽空き家の解体費補助(1/2・上限50万円)、平地への住み替え引越費補助(上限40万円)、学生居住支援(2012年〜)** / https://www.city.yokosuka.kanagawa.jp/4821/yatobank/test-top.html / ◎
- 2023年に「谷戸再生事業」で民間事業者2者を選定、民間共創型の再生に着手 / https://www.townnews.co.jp/0501/2023/11/10/705965.html / ○

## 2. 空き家統計

- **令和5年(2023年)住宅・土地統計調査: 総住宅数194,640戸のうち空き家30,270戸、空き家率15.6%**(前回比+1,520戸) / https://www.city.yokosuka.kanagawa.jp/0830/data/kekkakouhyou/documents/05juutaku.pdf / ◎
- **全国平均の空き家率は13.8%**。横須賀市は約1.8pt上回る / https://www.stat.go.jp/data/jyutaku/2023/pdf/kihon_gaiyou.pdf / ◎
- 市の空き家は1988年から30年で約2.8倍に増加 / https://www.townnews.co.jp/0501/2023/01/13/659693.html / ○
- 「第2期横須賀市空家等対策計画」(2022年4月) / https://www.city.yokosuka.kanagawa.jp/4821/akiyakeikaku/documents/plan2.pdf / ◎

## メタファ案

1. 「坂の上の空き家、ふもとの暮らし」— 玄関まで階段100段=エレベーターのないタワマン暮らし
2. 「7軒に1軒が留守のまち」— 空き家率15.6%の翻訳
3. 「まちの体も"血管"から老いる」— 上下水道・橋365橋は還暦世代のインフラ
4. 「Sランクの目覚まし時計」— 武山断層帯(30年以内6〜11%)
5. 「広げた手のひらを、もう一度握りこぶしに」— コンパクトシティ=店じまいではなく席替え
