# yokosuka-todo — 横須賀市が先送りせず解くべき課題

横須賀市の構造的な課題を、市民が読めるかたちで網羅的に整理・公開する静的サイト。

- 判断軸(ベンチマーク): **持続可能性をもった横須賀市の発展と市民のQOLの両立**
- 雛形(構成の参考): [japan-todo](https://fladdict.github.io/japan-todo)
- 公開: GitHub Pages(`main`ブランチ `/docs`)

## 各課題ページに書くこと

課題の定義 / データで見る現状 / 守りの一手 / 攻めの一手 / 進め方(短期・中期・長期) / トレードオフ / オススメの構成 / 実現方法(財源・体制・合意形成) / 出典

## 更新の流れ(Claude Code)

情報収集 → 分析 → 静的ページの更新 → ファクトチェック → インデックス更新 → アップロード

詳細は [CLAUDE.md](CLAUDE.md)(運用ルール)と [STYLE_GUIDE.md](STYLE_GUIDE.md)(文章ルール)を参照。

## ローカルで見る

ビルド不要。`docs/index.html` をブラウザで開くだけ。
